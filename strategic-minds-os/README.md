# Strategic Minds OS Sandbox

Controlled staging area for Strategic Minds OS prototypes.

## Allowed

- prototype pages
- static previews
- documentation
- build logs
- test reports
- pull requests
- Vercel preview preparation
- GitHub Pages preview preparation

## Not allowed without operator approval

- production deploys
- live Shopify publishing
- Supabase production migrations
- API secrets in files
- paid API usage
- client private data

## Source of truth

Google Sheet: `00 - SHOPIFY WORKFLOW Command Center`

Key tabs:

- 400_Intake_Build_Queue
- 414_Benchmark_Targets
